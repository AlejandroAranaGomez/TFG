Para poder ver las imágenes e información relativa a la API de los ejercicios de gimnasio debemos ejecutar en esta carpeta en la terminal este comando: "node server.js". Este pequeño programa crea una API REST para poder listar los 1000 ejercicios de gimnasio que están disponibles en la aplicación. Para ello se cogen los ejercicios del repositorio de GitHub de exercise-db, se crea el endpoint y se paginan los ejercicios. Por último, se arranca el servidor en el puerto 3000.
